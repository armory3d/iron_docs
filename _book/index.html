Introduction

Welcome

Iron is a library for building ultra-portable 3D tools using Kha and Haxe. Based on deployment requirements, JavaScript/C++ can be used. It is powering Armory, but is structured to run stand-alone.

While it is possible to operate Iron directly by hand, it is heavily data based. Instead, the library is designed to be integrated in custom tooling which will feed the data. Iron will handle full rendering and content pipelines, but will not provide visuals (no predefined shaders & render-path) or interaction.

Tooling:

  • Haxebullet for 3D physics
  • Haxerecast for 3D navigation
  • HaxeUI to build user interfaces
  • KodeStudio as IDE
